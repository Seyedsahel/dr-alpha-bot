# بخش یادآوری ترمیم

این بخش را خیلی حرفه‌ای می‌کنیم.

کاربر وارد می‌شود:

```
ثبت یادآوری ترمیم
```

سپس:

```
کدام خدمت را انجام داده اید؟
```

مثلاً:

```
بوتاکس
```

بعد:

```
تاریخ انجام خدمت را وارد کنید
```

مثلاً:

```
1405/03/15
```

ربات خودش محاسبه می‌کند:

```
زمان یادآوری:1405/06/15
```

و ذخیره می‌کند.
