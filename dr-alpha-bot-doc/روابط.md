Relationship ها

الان سه مدل داریم:

User
Appointment
AvailableSlot

و روابط اینطوریه:

User
 └── Appointments

AvailableSlot
 └── Appointment

Appointment
 ├── User
 └── AvailableSlot

یعنی:


هر User می‌تونه چند Appointment داشته باشه
هر AvailableSlot فقط یک Appointment می‌تونه داشته باشه
هر Appointment متعلق به یک User و یک Slot هست
